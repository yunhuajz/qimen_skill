# SKILL.md - 奇门遁甲排盘工具

## 描述

奇门遁甲排盘与分析工具 - 中国传统占卜术奇门遁甲排盘系统

## 版本

1.0.0

## 作者

Claude Skill Creator

## 功能

- 时家奇门排盘（九宫、天盘、地盘、人盘、神盘）
- 三种起局方法：拆补法（默认）、置闰法、茅山法
- 两种流派：转盘奇门、飞盘奇门
- 自动识别吉格凶格（20-30种常见格局）

## 使用示例

### 基本排盘

```typescript
import { paiPan, formatResult } from 'qimen-skill';

const result = paiPan({
  year: 2024,
  month: 12,
  day: 21,
  hour: 10,
});

console.log(formatResult(result));
```

### 指定起局方法

```typescript
const result = paiPan(time, {
  method: '拆补法',  // 或 '置闰法'、'茅山法'
  school: '转盘奇门', // 或 '飞盘奇门'
});
```

### 交互式使用

当你需要排奇门遁甲盘时，我会：

1. 询问具体的公历日期时间
2. 询问起局方法（拆补法/置闰法/茅山法）
3. 询问流派（转盘奇门/飞盘奇门）
4. 生成排盘结果并展示九宫格
5. 列出识别出的吉格和凶格

## 排盘结果解读

排盘结果包含：

- **局信息**：阴阳遁、局数、节气、元
- **旬首**：时辰的六甲旬首
- **值符**：值符星及其所在宫位
- **值使**：值使门及其所在宫位
- **九宫格**：每个宫位的神、星、门、天盘、地盘
- **格局**：识别出的吉格和凶格

## 支持的吉格

- 青龙返首、飞鸟跌穴
- 天遁、地遁、人遁、神遁、鬼遁
- 风遁、云遁、龙遁、虎遁

## 支持的凶格

- 伏吟、反吟
- 六仪击刑、门迫、六仪入墓
- 五不遇时
- 悖格、刑格
- 天乙伏宫、天乙飞宫
- 荧惑入白、白入荧
- 腾蛇夭矫、朱雀投江

## 依赖

- TypeScript 5.0+

3. 真太阳时计算暂未实现
4. 刻家奇门、日家/月家/年家奇门暂未实现
